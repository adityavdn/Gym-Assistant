## 📖 Overview

Web-based gym assistant application providing workout routines, exercise guides, progress tracking, and fitness tips. Interactive interface for personalized fitness planning.

## 🎯 Features

- Workout routine generator
- Exercise demonstrations
- Progress tracking
- Calorie calculator
- BMI calculator

## 🛠️ Tech Stack

- HTML5, CSS3
- JavaScript (ES6)
- Responsive Design

## 🚀 Usage

```bash
git clone https://github.com/adityavdn/Gym-Assistant.git
cd Gym-Assistant
# Open index.html in browser
```

## 📱 Features

- Custom workout plans
- Exercise library
- Goal setting
- Progress visualization
